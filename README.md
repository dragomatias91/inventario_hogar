# 🏠 Inventario del Hogar

Aplicación web desarrollada con Django para administrar el inventario del hogar. Permite gestionar productos y categorías de manera sencilla.

## 🛠️ Tecnologías
- Python
- Django
- Bootstrap 5
- SQLite

## 📦 Instalación

1. Clonar el repositorio
2. Crear y activar el entorno virtual
3. Instalar dependencias
4. Aplicar migraciones
5. Crear superusuario
6. Levantar el servidor

## 🚀 Orden de prueba

1. Entrar al admin en `/admin` y crear categorías
2. Desde el admin crear productos asignándoles una categoría
3. Ver el listado de categorías en `/categorias`
4. Ver el listado de productos en `/productos`

## 📁 Estructura

- `core` → home y template base
- `categorias` → modelo y vista de categorías
- `productos` → modelo y vista de productos
